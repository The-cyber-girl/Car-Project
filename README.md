# Car Project

This beginner-friendly Java project demonstrates Object-Oriented Programming (OOP) concepts using a `Car` class.

## Features
- `Car` class with attributes like brand, model, year, and color.
- Constructor to initialize car objects.
- Methods to display car information and update details.
- Demonstrates object creation, method calls, and managing object state.

## How to run
1. Compile `Car.java` and `Main.java`.
2. Run `Main.java` to see example car information displayed.

## Example Output
- Shows each car's brand, model, year, and color.
- Updates car details if needed.
- Prints information in a clear format.
